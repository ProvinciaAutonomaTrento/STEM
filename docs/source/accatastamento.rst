Accatastamento
================================

Il modulo esegue l'accatastamento di più immagini.

.. image:: accatastamento.png


Input
------------
**Dati di input**: nella finestra compaiono i raster attualmente aperti in QGIS. Selezionare i raster da accatastare.


Parametri
------------

* **Formato di output**:
	* *ENVI*: formato compatibile con il software ENVI.
	* *GTIFF*: formato GeoTiff.

Output
------------
* **Risultato**: inserire percorso e nome del file di output.

Opzioni
------------
* *Esegui localmente*: selezionare per eseguire in locale.
* *Raster muiltibanda*: selezionare se l'output è multibanda
* *Aggiungi risultato alla mappa*: selezionare se il risultato va aggiunto alla mappa
